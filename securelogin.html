Background



Secure Login is an multi-factor authentication flow meant to be implemented as an out-of-the-box solution in all Citi mobile apps (both iPhone and Android). The solution includes methods such as PIN code,

Face Recognition and Fingerprint Recognition.

Challenge

The challenge was to create a native login experience for both Android and iPhone, that can be used as an out-of-the-box product and yet customizable enough to feel native of each Citi app. The flows must be short and simple, while leaving the option to add more authentication methods in the future.

My Tasks

Planning and Scope Definiton

I was in charge of understanding all the possible options and flows, assuring the user can configure more than one method of authentication and giving a proper solution to any possible pitfall.


Adaptation to Android and iOS

The flows were to be implemented in both Android and iOS platforms, so for each the flow should feel as seamless and native as possible.


Interactive Prototyping and Specs

I executed detailed specs, wireframes and prototypes to address any stakeholder concern, by showing the flow in a more holistic,

understandable manner.


Supervising of Visual Design Mockups

I supervised a UI designer and directed the style of the mockups. We agreed on a style that should be easy to adapt to different apps which have their own styleguide. Therefore, I put accent on having components that would be flexible on their color and font styles.
