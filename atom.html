Atom Deploy

DESKTOP WEB APP, CITI INNOVATION LAB 2017

Background



Atom is the platform for applications version management at Citi, used by developers and release managers.


The system allows to manage app versions, e.g. which versions goes to which user, creating beta testers and overall management of versions.

Challenge

The existing version is very limited and lacks any basic information architecture. I was required to create a new structure for the system which would organize the elements in an intuitive way that supports the developers and release manager's workflow.


I worked intensively with the product manager to understand the relation between the elements, and then choose a structure that would support the required actions.
