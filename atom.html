About

Background



Atom is the platform for applications version management at Citi. This platform is fully developed at the Lab and used by a thousand developers at Citi around the globe. The system allows to manage which app version goes to which user, creating beta testers and overall management of versions.

The current version is very limited and lacks a clear information architecture.

Challenge

I was required to create a new structure for the system which would organize the elements in an intuitive way that supports the developers and release manager's workflow.

I worked intensively with the product manager to understand the relation between the elements, and then choose a structure that would support the required actions.
